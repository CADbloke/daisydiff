<p class="note">The Daisy API for the latest version of the Daisy CMS – as of
this writing version 2.0 – can be found at
<a href="http://cocoondev.org/javadoc/daisy/2.0/">http://cocoondev.org/javadoc/daisy/2.0/</a>
</p>
 
<p>Everyone who seriously wants to extend or modify daisy in some way or the
other, using the Java programming language needs access to the Daisy's javadoc.
Since I make use of it quite often, I put it online
<a href="http://daisyapi.deininger.net">here</a> (Daisy 1.5-dev), others may
benefit from that, too.</p>
