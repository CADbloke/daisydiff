<h1>Students</h1>

<p>What does that mean? Well, if you are a student, and your proposal qualifies,

<p>Much like a source code version control system, in Daisy a version history of

useful. Currently the diffs shown in Daisy are based on the HTML source text
(just try Page Action -&gt; Versions in the menu on top of this page to see an
example). It would be very nice to have graphical diffs instead, thus simply a
HTML document with changes, additions and deletions marked (a bit like the
